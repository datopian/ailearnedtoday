## Overview

Google's "vibe design" tool that turns natural language into high-fidelity UI designs. Features an AI-native infinite canvas for exploring ideas from early ideations to working prototypes.

## 5 Major Upgrades

🎨 **AI-Native Canvas** - Infinite canvas with context from images, text, or code  
🧠 **Smarter Design Agent** - Reasons across entire project evolution; Agent manager tracks parallel work  
🎤 **Voice** - Real-time design critiques and updates via voice commands  
⚡ **Instant Prototypes** - "Stitch" screens together, auto-generate logical flows, click "Play" to preview  
📐 **Design Systems and DESIGN.md** - Agent-friendly markdown for exporting/importing design rules across tools

## Key Features

- **DESIGN.md**: Agent-friendly markdown file for design systems that works across design and coding tools
- **Voice commands**: Real-time design critiques, interviews, live updates ("give me three different menu options")
- **MCP integration**: Bridge to other tools via MCP server, SDK, and skills
- **Exports**: Developer tools like AI Studio and Antigravity
- **Interactive prototypes**: Auto-generate user journeys, preview app flows instantly
