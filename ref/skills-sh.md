## Overview

The Agent Skills Directory aggregates skills from major repositories and community sources. Acts as a search/discovery hub for agent capabilities.

**Sources include**:
- microsoft/github-copilot-for-azure (2.4M total)
- microsoft/azure-skills (1.6M total)

- pbakaus/impeccable (155.7K - 177.6K total)
- coreyhaines31/marketingskills (115.4K total)
- supercent-io/skills-template (44K - 54K total)

**Think**: npm registry for agent skills — central place to browse what exists before building your own.
