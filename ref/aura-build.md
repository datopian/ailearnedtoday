# Aura (aura.build)

AI website builder and DESIGN.md generator for learning and reusing design systems.

## Overview

Aura is an AI-powered website builder by Meng To that can analyze any URL, generate a **DESIGN.md** describing its design system, and recreate the site layout very accurately. The generated prompts can be reused as a starting point to:

- Learn real-world design systems
- Stay visually consistent across pages
- Avoid generic "purple gradient" aesthetic

It fits into the emerging pattern of **design system codification** — turning visual decisions into structured text specs that agents and humans can reuse.

## Key Capabilities

- **DESIGN.md from any URL**: Given a target site, Aura extracts typography, colors, spacing, component patterns, and layout into a structured markdown spec.
- **Recreate site layout**: Uses the extracted design system to rebuild the site fairly accurately.
- **Prompts as assets**: The DESIGN.md + prompts become reusable assets for:
  - Teaching AI models a design system
  - Onboarding new designers/developers
  - Applying the same system to new pages and products
