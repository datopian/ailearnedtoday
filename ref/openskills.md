# OpenSkills

Universal skills loader for AI coding agents — npm package for programmatic skill loading.

![OpenSkills GitHub](https://screenshotit.app/https://github.com/numman-ali/openskills)

## Links

- **GitHub**: https://github.com/numman-ali/openskills (9.2k ⭐)
- **Install**: `npm i -g openskills`

## Overview

OpenSkills is a universal skills loader that treats agent skills like npm dependencies. Install globally, import programmatically, manage via CLI.

**Philosophy**: Skills as code dependencies — version, install, import like any other package.

**For developers who want**: CLI-first workflow, programmatic access, package-manager-style skill distribution.
