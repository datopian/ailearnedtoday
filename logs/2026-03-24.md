# 2026-03-24 Open-Source Agents and Skills Management Tools

## Added

- [[deerflow]] - ByteDance's open-source SuperAgent harness (41.1k ⭐): research, coding, creation with sandboxes, skills, memory, subagents in Docker environment.
- [[openclaw]] - Personal AI assistant (333.4k ⭐): 24/7 assistant via messaging apps with persistent memory, skills, sandbox, self-hackable, MIT license.
- [[skills-managers]] - Tools for discovering, organizing, and distributing agent skills across projects: skills.sh (registry), Oh My Agents (local sync), OpenSkills (npm-style).
- [[oh-my-agents]] - Local workspace for managing prompts/skills across agents: discover, organize, deploy with preview, bidirectional sync.
- [[skills-sh]] - Vercel's Agent Skills Directory: central registry aggregating 2.4M+ skills from major repos for discovery.
- [[openskills]] - Universal skills loader (9.2k ⭐): npm package for programmatic skill loading, CLI-first, treats skills as code dependencies.
