# 2026-02-23 Design Repo Pattern for AI-Friendly Workflows

## Design Repo Pattern (Draft Post)

Draft post outlining a pattern for keeping design work, inspiration, and exploration separate from the main application repo. Create a dedicated design repo with `inspiration/` (screenshots + examples organized by topic), `designs/` (HTML prototypes), `notes/` (design principles, color palettes), and `assets/` (screenshots). Benefits: keeps main repo clean, AI-friendly (agents can read and reference entire design context), version-controlled, not locked to proprietary tools like Pinterest. See: `posts/2026-02-23-design-repo-pattern.md`
