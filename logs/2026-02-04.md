# 2026-02-04

## [[msgvault]] - Local-first Email Archive

Wes McKinney (pandas/Arrow creator) built msgvault: local-first email archive powered by DuckDB with MCP server for AI agent integration. Handles 20 years of Gmail (2M emails, 150K attachments) in a single Go binary.

## [[inference-sh]] - The Agent Runtime

Production runtime platform for AI agents. Provides durable execution, tool orchestration (150+ apps), observability, and agent primitives (deep-agents, human-in-the-loop, widgets). Pay-per-execution, self-hostable, integrates with x402 payment protocol.

## [[pi.dev]] - New Home for pi Coding Agent

Mario Zechner announces pi coding agent now has official home at https://pi.dev/ (domain donated by exe.dev). Minimal, extensible terminal coding harness - build your workflow with TypeScript extensions, skills, packages. 15+ providers, tree-structured sessions.
