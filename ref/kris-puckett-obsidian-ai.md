## The Setup

Kris Puckett switched to Obsidian for docs/data and used Claude Code to:
1. Build custom iOS app connecting Apple Health to Mac mini via CLI and local network
2. Refactor Obsidian theme and layout
3. Incorporate health visualization in daily briefing

## Why This Matters

Demonstrates how AI coding agents enable non-developers to build deeply personalized integrations with markdown-based knowledge systems. The combination of:
- Obsidian's markdown foundation
- Claude Code's ability to build custom tools
- Local network + CLI integrations
- Personal health data visualization

Results in bespoke knowledge infrastructure that would traditionally require hiring developers or using rigid SaaS products.
