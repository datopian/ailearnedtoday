## [[obsidian-claude-code-personal-os]] - Building a 24/7 Personal Operating System with Obsidian + Claude Code

Greg Isenberg's workflow for using Obsidian (markdown notes) + Claude Code to build a personal operating system where agents read your entire knowledge vault, understand context across sessions, and act on your behalf. Key insight: "Markdown files are the oxygen of LLMs" — write everything in interconnected markdown, install Obsidian CLI for agent access, build custom slash commands (/context, /trace, /connect), let agents surface patterns. Human writes vault, agents read/suggest/execute.

## [[2028-global-intelligence-crisis]] - Speculative Scenario on AI-Driven Economic Collapse

CitriniResearch macro memo written as if from June 2028, modeling "Global Intelligence Crisis" where rapid AI advancement triggers mass white-collar unemployment, mortgage crisis, and economic collapse. Key mechanism: "Intelligence Displacement Spiral" with no natural brake — AI improves, companies cut workers, displaced workers spend less, companies invest savings into more AI, repeat. Not a prediction, but underexplored left-tail risk scenario.
