## [[codexbar]] - macOS Menu Bar App for AI Coding Agent Limits

Free, open-source macOS menu bar app by Peter Steinberger that tracks session and weekly usage limits for multiple AI coding agents (Codex, Claude Code, Cursor, Gemini, Antigravity, Droid, Copilot, z.ai) — shows real-time bars, reset countdowns, credits so you know when you're safe to ship. Reuses browser cookies for dashboard access, includes CLI for automation.
