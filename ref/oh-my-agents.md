# Oh My Agents

Local workspace tool for managing prompts and skills across multiple AI agents.

![Oh My Agents](https://screenshotit.app/https://oh-my-agents.app)

## Links

- **Website**: https://oh-my-agents.app
- **Tagline**: "One Source. Every Agent. In Sync."

## Overview

Oh My Agents helps you discover, organize, deploy, and sync prompts and skills across all your AI coding agents and projects. Central asset library with distribution rules, preview-before-deploy, and bidirectional sync.

## Workflow

1. **Discover**: Auto-detect agents, scan project directories, import from GitHub or local folders
2. **Organize**: Central workspace with search, categorization, per-project/per-agent rules, custom aliases
3. **Deploy**: Preview sync plan, one-click distribution to target locations, conflict detection
4. **Evolve**: Pull changes back from projects to central library, update all linked projects at once

**Key insight**: Treats prompts and skills as evolving assets that need version control and distribution across your ecosystem.
