## Key Themes

### 1. Agentic Engineering

Karpathy has identified "agentic engineering" as an emerging discipline - the practice of:
- Setting up long-running orchestrator agents
- Managing multiple parallel code instances
- Ascending layers of abstraction
- Providing high-level direction while agents handle implementation

### 2. Practical AI

Focus on what actually works in production:
- Real-world examples (home video dashboard setup)
- Honest assessment of limitations
- Emphasis on verifiable functionality
- Building intuition for task decomposition

### 3. Educational Clarity

Known for:
- Making AI accessible to practitioners
- Concrete examples over abstract theory
- Open-source educational tools
- Transparent discussion of both capabilities and constraints
